                                                                 *testfile.nvim*

This plugin will create a buffer pointing to a testfile for the current buffer

                                                        *testfile.create()*

testfile.create()

    The current buffers name, and the config for its file extension, will be used to create a new buffer with a test path. >

        testfile.create()
<
                                                           *testfile.toggle()*

testfile.toggle()

    Toggle between a test buffer or srfile buffer.
    The current buffers name, and the config for its file extension, will be used to create or switch to its source/test buffer equivalent. >

        testfile.toggle()
<
